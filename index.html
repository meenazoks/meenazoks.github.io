Brick Out

Description: Brick Out is a game consisting of a ball, a paddle, and blocks. The ball bounces off the paddle and destroys any blocks it hits. The goal of the game is to destroy all blocks. The paddle is controlled by the trackpad, and if the user misses the ball, the paddle and ball are reset.

Class: Computer Science 211 -- Fundamentals of Computer Progamming II

Language: C++

Garden

Description: Garden runs a simulation to represent the state of a garden everyday.

  • If a flower has only one or no neighbors, it dies of loneliness.
  • If a flower has four neighbors, it dies of overcrowding.
  • Empty locations that have exactly three neighbors will grow a flower.
Any cell that undergoes a change during the current step will have a red border.

Class: Computer Progamming III (high school)

Language: Java (focus on GUI)
